# Badge

A badge component for displaying status, labels, or small pieces of information.

## Usage

```jsx
import Badge from '@workpace/design-system';

<Badge variant="success">Completed</Badge>
<Badge variant="warning" size="sm">Pending</Badge>
<Badge variant="error" size="lg">Error</Badge>
```
